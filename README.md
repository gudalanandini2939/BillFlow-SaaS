# BillFlow SaaS

BillFlow SaaS is a full-stack MERN application developed for managing invoices and billing operations.

## Features

* User Registration
* User Login
* JWT Authentication
* Create Invoices
* View Invoices
* Edit Invoices
* Delete Invoices
* Protected Routes
* MongoDB Database

## Tech Stack

### Frontend

* React.js
* React Router DOM
* CSS
* Vite

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Authentication

* JWT
* Bcrypt.js

### Deployment

* Vercel
* Render
